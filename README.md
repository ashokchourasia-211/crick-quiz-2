# Cricket Quiz

A simple and interactive quiz application to test your knowledge of cricket.

## Description

This project is a web-based quiz that presents users with a series of multiple-choice questions about cricket. The user's score is tracked, and they receive immediate feedback on their answers.

## Setup

To run this project locally, simply clone the repository and open the `index.html` file in your web browser.

```bash
git clone https://github.com/your-username/cricket-quiz.git
cd cricket-quiz
# Open index.html in your browser
```

## Usage

1.  Open the `index.html` file in a web browser.
2.  The first question will be displayed with four options.
3.  Click on the option you believe is correct.
4.  You will receive immediate feedback, and the correct answer will be highlighted.
5.  Click the "Next" button to proceed to the next question.
6.  Your score will be updated at the bottom of the quiz.
7.  At the end of the quiz, your final score will be displayed.
